fastaContainerFactory¶

Usage¶

import pyproteinsExt.fastaContainer as fasta
fastaContainer = fasta.parse("your_file.fasta")

You can iterate through fastaContainer

for f in fastaContainer:
   print(f.prot)
   print(f.seq)
   print(f.header)

second_entry = fastaContainer.entries["tr|A0A2V4DJ72|A0A2V4DJ72_9EURY"]
print(second_entry.seq)
